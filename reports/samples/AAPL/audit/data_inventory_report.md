# Data Inventory Report

## What Was Captured

- Company profile: true
- Price history: true
- Income statement: true
- Balance sheet: true
- Cash flow: true
- Valuation snapshot: true
- Segment data: false

## Used In Charts

- Price history feeds Figure 1 and Figure 2 when available.
- Income statement feeds Figure 3 when revenue data is available.
- Cash-flow data feeds Figure 4 when operating cash flow / capex data is available.
- Valuation snapshot feeds Figure 5 when meaningful valuation fields exist.

## Used In Tables

The report status, financial snapshot, money-flow summary, research blueprint, and data-gap tables use compact values and interpretation text. Raw CSV-style data is not pasted into the report body.

## Missing Critical Data

No critical data gaps were flagged by the current blueprint.

## Unused Data Policy

Text fields are explained in prose/dashboard cards. Missing or non-visualizable data is shown as a data gap instead of being forced into an empty chart.
