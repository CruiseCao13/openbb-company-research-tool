# Report Language Style

Write like a careful research memo, not generic AI prose.

Prefer:

- The key issue is...
- This matters because...
- The data supports X, but not Y.
- This report cannot verify...
- The next manual check is...

Avoid:

- strong potential
- investors should pay attention
- opportunities and challenges
- future remains uncertain
- based on the data, we can see

Chinese reports should be concise and natural, not line-by-line translations from English.
