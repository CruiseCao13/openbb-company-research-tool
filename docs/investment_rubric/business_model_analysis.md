# Business Model Analysis

Checklist:

- Identify what the company sells before choosing a research frame.
- Separate product revenue, service revenue, contract revenue, financial spread income, rent, regulated utility revenue, and one-time revenue.
- Name what the company is not when common wrong frames are plausible.
- If the provider description is thin, use an unknown/data-limited frame rather than inventing revenue engines.
- A business model claim needs provider description support, segment data, or an explicit data-gap boundary.
