# v5 Investment Rubric Library

This folder contains user-owned research principles for v5 quality training. It is not copied from copyrighted books. It is a compact checklist library used to guide prompts, validators, quality scoring, and training-case review.

Use rule:

- Retrieve only the relevant rubric for the company and issue type.
- Do not paste the whole library into every AI prompt.
- Prefer locked data and provider descriptions over generic sector assumptions.
- If data is missing, mark a data gap instead of inventing a conclusion.
