# Red Flags

Red flags must name a mechanism:

- Wrong framework risk.
- Revenue engine not supported by provider description.
- Negative operating cash flow with high capex or R&D.
- Debt refinancing pressure.
- Dilution risk.
- Customer concentration.
- Data gaps in critical fields.
- Unsupported numeric claims.
- Report status PASS despite hard failure.

Avoid vague labels such as "uncertainty" unless the source is named.
